A cidade inteira na palma de suas mãos!


Viva a experiência de um hacker de cinema nesse jogo enquanto exercita tópicos introdutórios de programação. Agir rapidamente e estrategicamente é crucial para ajudar seus parceiros numa jornada para se tornar o maior Hacker do mundo!

- Abordagem de ensino de Introdução à Programação através de jogos
- Uma aventura interativa através de 3 capítulos
- Compreenda conceitos de fluxo de programa, lógica de condicionais, laços, variáveis, tipos de dados, funções, números binários e hexadecimais
- Ou então, jogue para se divertir (Não é necessário nenhum conhecimento prévio!)
